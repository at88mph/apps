Download a text file containing a list of URLs that can be consumed with a script or directly with the wget command. Each line contains an URL or, when URL generation fails, an error message of the form: ERROR\t<Error message>

wget command to download all the URLs in the above text file:
% wget --http-user=CADC_USER --http-password=CADC_PASSWORD -i FILE_NAME

Be certain to fill in your own CADC_USERNAME, CADC_PASSWORD and the used FILE_NAME (default: cadcUrlList.txt) in the appropriate places in that command or you will get an error from wget. See below for more wget options.
