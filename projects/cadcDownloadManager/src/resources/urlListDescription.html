Download a manifest text file containing a list of URLs that can be consumed with a script or directly with the wget command. There are two formats for a row in this file:

OK\t<URL>
ERROR\t<Error message>

The following wget command downloads all the URLs in the manifest file:
% wget --http-user=CADC_USER --http-password=CADC_PASSWORD -i FILE_NAME

Be certain to fill in your own CADC_USERNAME, CADC_PASSWORD and the used FILE_NAME (default: cadcUrlList.txt) in the appropriate places in that command, or you will get an error from wget.
